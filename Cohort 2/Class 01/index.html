  • How do you learn how to learn?
  • Active Recall
  • Spatial Repetittion
  • What is Active Recall?
  • Actively retrieving information from memory
  • boosts long-term retention and learning effectiveness through engagement
  • What is Spaced Repetition?
  • Repeating information in a specific arrangement or pattern
  • enhances memory and learning through pattern recognition and reinforcement
  • What is the internet?
  • A global network connecting millions of devices, allowing communication, information sharing, and access to online resources
  • Connects two types of computers:
    • client devices
    • servers
  • Whats a client device?
  • Any device a person can use to access the internet
  • What is a server?
  • A computer that provides services or resources to client devices
  • It returns files like HTML, CSS and JavaScript that can be read by the client device and produce a visual website on a client device
  • What is HTML?
  • Stands for "Hypertext Markup Language"
  • Client-side, or front-end code that you see
  • Pretains to the content of a website
  • content / text / images / video / articles
  • What is CSS?
  • Stands for "Cascading Style Sheets"
  • Client-side, or front-end code that you see
  • Pretains to how the content you see is styled (presented)
  • color / style / positioning
  • What is JavaScript?
  • A programming language used for web development, providing interactivity and dynamic content on websites
  • Pretains to the functionality of the elements in an HTML doc
  • drag and drop / click / mouseover
  • Whats a Front-End web developer do?
  • Work with HTML, CSS & JavaScript to create content on user devices
  • Uses "Client-Side" Code
  • What does a Back-End developer do?
  • Works with code that stores information to a database and stores it in a server
  • Users "Server-Side" Code
  • background data processing
  • What is a Full-Stack Web developer?
  • A software engineer that works on both the client and server side of a application
  • Works on front end and back end
  • What is an ISP?
  • Internet Service Provider
  • We pay to them to connect us to the internet
  • How responses between computers get back and forth
  • What is a DNS?
  • Domain Name Server
  • A Fancy Address Book
  • Takes the URL address and provides user with an IP address
  • What is a URL?
  • Uniform Resource Locator
  • The address listed in the DNS
  • What is an IP Address?
  • Internet Protocol Address
  • A numerical representation of a URL
  • How does a URL produce a website?
  • When the URL is typed in, the request makes its way to a DNS. The DNS then finds the URL's IP address, and uses that IP address to locate the server that is storing the URL request data and returns it to the device making the request.
  • What is HTTP?
  • Hypertext Transfer Protocol
  • The process of a device making and processing a site request
  • Has fallback of having data trail be seen by anyone watching your IP activities
  • What is HTTPS?
  • Hypertext Transfer Protocol Secured
  • Allowed for HTTP to continue but with more security measures for the user
  • What is a VPN?
  • Virtual Private Network
  • Encripts your request and sends it to a server for you, receives the requests and returns it to you
  • What is TOR?
  • The Onion Router
  • Free and open-source software for enabling anonymous communication
  • Directs Internet traffic via a free, worldwide, volunteer overlay network that consists of more than seven thousand relays
  • What is HTML5?
  • A series of recommendations added to the standard HTML language to make HTML more uniform and reduce site breaks across devices
  • What is W3C?
  • The World Wide Web Consortion
  • The think tank headed by Tim Berners Lee to develop the standards for HTML and its following iterations such as HTML5
  • Introduced new features including:
    • Video Files
    • Audio Files
    • Canvas (allows you to draw and render graphics directly on a web page)
    • Geolocation
    • Websockets
  • What is the "Golden Rule" of development?
  • Keep a seperation of concerns, meaning:
    • Keep only content in HTML
    • Keep only styling in CSS
    • Keep only functionality/behavior in JavaScript
  • This will help with:
    • Content Organization
    • Content Readablity
    • Simplifies Debugging

  • How do you learn how to learn?
  • Active Recall
  • Spatial Repetittion
  • What is Active Recall?
  • Actively retrieving information from memory
  • boosts long-term retention and learning effectiveness through engagement
  • What is Spaced Repetition?
  • Repeating information in a specific arrangement or pattern
  • enhances memory and learning through pattern recognition and reinforcement
  • What is the internet?
  • A global network connecting millions of devices, allowing communication, information sharing, and access to online resources
  • Connects two types of computers:
    • client devices
    • servers
  • Whats a client device?
  • Any device a person can use to access the internet
  • What is a server?
  • A computer that provides services or resources to client devices
  • It returns files like HTML, CSS and JavaScript that can be read by the client device and produce a visual website on a client device
  • What is HTML?
  • Stands for "Hypertext Markup Language"
  • Client-side, or front-end code that you see
  • Pretains to the content of a website
  • content / text / images / video / articles
  • What is CSS?
  • Stands for "Cascading Style Sheets"
  • Client-side, or front-end code that you see
  • Pretains to how the content you see is styled (presented)
  • color / style / positioning
  • What is JavaScript?
  • A programming language used for web development, providing interactivity and dynamic content on websites
  • Pretains to the functionality of the elements in an HTML doc
  • drag and drop / click / mouseover

Definition / Details

  • CSS wtitten in the HTML file that provides styling for content "above the fold" of a website
    • folld is the height of 100% of the device viewport
  • How do you link tour CSS file to your HTML?
  • What is the context of a CSS elements?
  • Entire element is called a "rule"
  • A rule is composed of a "selector" and a "declaration block"
  • A "selector" is the tag, class, or id of an element that exists in the HTML document
  • A 'Declaration block" is the grouping of a selectors applied "declaration" within a set of curly braces
  • A "declaration is the pairing of a "property" and "value" that will be applied to a selector
  • A "property" is a pre-existing element within CSS that allows for an elements content to be manipulated and altered based on the effects the property can do
  • A 'value' is the associated utility of a property, used in styling to be applied to a selector.
  • What's the cascade?
  • Tags can be overwritten based on the order in which they are written in the CSS file
  • "The last to be applied is what gets applied"
  • "The higher the specificity, the harder to override
  • Whats specificity?
  • The value of a selector determined by the tag, class, and id total
  • tags add 1 point, class adds 10 points and id's add 100 points
  • classes can be reused on multiple elements while id's must be unique and can only identify one element
  • adding (!) important adds 1000 points of specificity and should be used only in emergencies
  • How do you identify color in CSS
  • Hard coded words like red and blue
  • RGBa values that accept values between 0 - 255 to represent red blue and green respectively to create shades of color and using the alpha to determine opacity
  • HSLa values that accept 0-255 for a hie, 0-100% for saturation and light, and using the alpha to determine opacity
  • HEX (hexidecimal) values: a 3 - 6 digit value representing the red, blue and gren values of a color, indicated by using the octothorpe (#)
  • How do you get fonts?
  • Go to "Google Fonts" and either:
    • import the file into your CSS directly
    • link the font file to your HTML
  • Add not only the font choosen, but a back up font and a basic font to ensure if a machine that can not render your choosen font, there is still an option for styling
  • How to use selector relationships in CSS?
  • the "parent > child" relationship illustrates that a direct decendant (child) is being targeted within another element (parent) and only within the parent.
  • the "parent child" relationship illustrates a non-direct decendant (child) being targeded within another element, reguardless of its nesting within othere elements
  • the "sibling + sibling" relationiship ilustrates a lateral relationship of two elements with the same parent, targeting elements immediately next to each other in the specified order
  • What is the box model?
  • The concept that everything in HTML is a box, and the logic behind how these boses take up space on a webpage
  • Every element has the capacity to expand its margin, border, and padding to increase the complete size of the content box taking up space on the webpage
  • said expansions can be uniform across the entire element, or be selective to the left, right, top, or bottom
  • Whats margin?
  • a CSS porperty used to move elements around on a page
  • Sets the space between itself and other elements on the page
  • What is border?
  • The defineable line between the edge of an element and the neginning of this margin space
  • What is padding?
  • The whitespace between the content of an element and the outlining border

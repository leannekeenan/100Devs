  • What is the "Semantic/Content Reasoning behind using the "paragraph" tag?
  • The paragraph tag is used for large blocks of text
  • la a block element in CSS meaning each new paragraph breaks to its own line instead of sharing line space
  • What is the "Semantic/Content Reasoning behind using the "span" tag?
  • The span tag is used for short text content
  • The span tag is a inline element, meaning it remains inline with other objects, sharing line space instead of breaking to a new line.
  • spans can be good for including text within text that is not necessarially an anchor
    • something like this
  • What is the "Semantic/Content Reasoning behind using the "pre" tag?
  • This is depreciated, but good to know if you are working with legacy code
  • Preserves whitespace in the code you write
    • This should be done in CSS
  • What is the "Semantic/Content Reasoning behind using the "anchor" tag?
  • The anchor tag is used to create links within the page to new content
  • anchor tags can access content using https to connect to outside content pages, or .html/.pdf/etc. for internal documents within the porgram file
  • What is the "Semantic/Content Reasoning behind using the "break" tag?
  • Used to add break lines between content
  • What is the "Semantic/Content Reasoning behind using the "em" tag?
  • Expresses SEO importance
  • Provides emphasis
  • Makes content italicized
  • What is the "Semantic/Content Reasoning behind using the "strong" tag?
  • Expresses SEO importance
  • Provides importance
  • Makes content bold
  • What is the "Semantic/Content Reasoning behind using the "ordered list" tag?
  • Creates a list with ordered steps
  • Adds numerical bulletpoints butting steps in a specific order
  • What is the "Semantic/Content Reasoning behind using the "unordered list" tag?
  • Creates a list in no specific order
  • Uses bulletpoints as list item indicators, but no order is specified
  • What is the "Semantic/Content Reasoning behind using the "divider" tags?
  • The divider (div) container is a non-semantic container
  • What is the "Semantic/Content Reasoning behind using the "section" tags?
  • Used to group like content
  • What is the "Semantic/Content Reasoning behind using the "article" tags?
  • Containing element used for things meant to be shared that do not require additional context
  • think blog posts, news articles, etc
  • What is the "Semantic/Content Reasoning behind using the "aside" tags?
  • For insulary/extra content like ads and banners
  • replaced sidebars
  • What is the "Semantic/Content Reasoning behind using the "header" tag?

  • A theory stating the majority of your code should be content, followed by styling presentsation using moderation, and minimal, necessary bahavior to make the content engaging or useful.

  • CSS Syntax
  • Whats the Cascade
  • Tha ability for a selector of a higher specificity or written in the CSS document after a previously declared selector can overwrite said previously defined selector
  • What are the three common relationship selector functions?
  • parent > child: specified direct children of a parent element
  • child parent: specified non-direct element within a parent element
  • sibling[specified] + sibling[element previous]: specifies an element with a sibling above it of the declared type
  • Whats an ID
  • a single use identifier for a DOM element
  • cant be reused or declared elsewhere in the code
  • gives 100 points of specificity
  • Can have only one
  • Whats a class
  • a multi use identifier for DOM elements
  • Can be reused
  • Can have more than one
  • gives 10 points of specificity
  • What are tags
  • The predefined elements in the DOM including span, p, img, etc.
  • gives 1 point of specificity
  • What is the important function
  • "!"
  • Used to define high importance to CSS elements
  • Gives 1000 points of specificity
  • Whats the box model
  • The theory every element in web dev is a box which can be manipulated by using its content height & width, padding, border, amd margin
  • Whats a float
  • a property trying to get eleents to the upper right corner as much as possible

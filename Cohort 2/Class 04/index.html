Definition / Details

  • What is the internet?
  • A wire
  • How different devices are connected
  • Client devices make requests, servers fulfill the requests
  • Back-End Developer
  • Writes code that enables servers to listen for requests from other devices and generate responses
  • Front-End Developer
  • Writes code that runs on client side devices
  • Full-Stack Developer
  • A developer that writes code for both front and back end fluently
  • Golden Rule
  • "The Seperation of concerns"
    • Keeps content in HTML
    • Keeps style in CSS
    • Keeps behavior in JavaScript
  • Keeps content organized for self and developer team
  • What is Progressive Enhancement?
  • A organizational idea that states the bulk of code should be the content (HTML), followed by a generous but not excessive amount of styling (CSS) and a minimal but necessary amount of functionality (JavaScript)
  • This is intented to keep the page from being overloaded when rendered, particularly on low broadband devices
  • Why is progressive enhancement important?
  • Helps with accessability (Hearing/Visual Impaired)
  • Helps process pages even without high-speed internet

  • To maintain Seperation of Concerns

  • What is critical Path CSS?
  • The CSS you absolutely need to load "above the fold" of a website
  • What is the fold of the website?
  • The fold is where the content of the erbsite meets the bottom of the devide it is viewed on
  • How do you link your CSS page to your HTML page

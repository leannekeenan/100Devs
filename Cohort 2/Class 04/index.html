  • What is the cascade?
  • The ability for a CSS rule to be overritten if :
    • The specificity value of the selector is higher than its previous rule
      • h1 {
        color: red
        }

        body main section h1 {
        color: blue
        }

      • This example, while stating the color of the h1 element will be red, it is overwritten and will now be blue as the specificity value is higher on "body main section h1" that "h1" alone

      • or

    • The order in which the rules are written in the CSS document can overwrite a rule
      • h1 {
        color: red;
        }

        h1 {
        color: blue;
        }

      • This example, while orignally setting the value of h1 ' color to red is overwritten to blue because a second command on the seame selector has been written after, leaving it as the last command to effect the selector
  • Essentially, the rules that come after can override rules that came before

  • How do you target specific elements?
  • Select by relationship
    • Direct Child (parent > child)
      • targets element (child) within another (parent) that are direct decendants
    • parent child
      • targets elements (child) inside another (parent) that may not be a direct decendant (nested inside another element)
    • sibling + sibling
      • targets elements that has the specified immediate sibling element
      • for example if a div has 2 paragraphs, and you use p + p in your CSS you will be targeeting the second paragraph only
  • Use element class or id
  • What is a class
  • a general identifier that can be used to target multiple elements at once
  • uses period(.) to identify class in CSS
  • Since it cn be used on multiple elements, it does not have to be expressly unique for each element
  • you can have as many classes as you wish per element
  • What is an Id
  • a way of uniquely identifying an element
  • names must be unique an ddistinct from any other keyword and can not be used elsewhere to define anything else
  • use octothorpe (#) to identify id's in CSS
  • you should have only one id per page element
  • What is specificity?
  • The value of an element based on its class id and tag cumulative value
  • tags add 1 points
  • class adds 10 points
  • id adds 100 points
  • important (!) adds 1000 and should be used for important items only
